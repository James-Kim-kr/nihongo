# 런치 체크리스트

- 접근성 감리: 스크린리더, 키보드 내비게이션, 명암/색상 대비, 캡션/대체 텍스트
- 데이터 보호 영향평가(DPIA): 데이터 최소화, 보존 기간, 권리 행사 절차, 감사 로그
- 아동/학생 보호: COPPA(<13), FERPA/GDPR, 학부모 동의 플로우 및 기록
- AI 안전 레드팀: 금지 주제/우회, 환각 대응, 연령 적합성, 오남용 신고·차단
- 성능/가용성: 초기 로드 < 3s, 상호작용 < 100ms, 오류 예산/SLA, DR/백업 복구
- 보안: 비밀/키 관리, 전송·저장 암호화, 취약점 스캔 및 패치, 침해 대응 계획
- 콘텐츠 품질: 표준 맵핑 검수, 루브릭/해설 품질, 저작권 확인, 메타데이터 정합성
- 데이터/분석: 핵심 이벤트 스키마, 대시보드, 품질 경보 임계치 설정
- 통합: LTI 1.3, SSO(SAML/OIDC), SIS 연동 테스트 및 샌드박스 검증
- 운영: 피처 플래그 전략, 롤아웃 계획(알파/베타/GA), 교사 연수/가이드 준비
- 지원: 헬프센터, SLA·SLO 명시, 티켓 흐름, 우선순위 규칙, 취약 사용자 보호 절차
